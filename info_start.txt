----- ЗАПУСК СКРИПТА -----

1. Сначала нужно получить список камер
Для запуска в терминале ввести:
python3 get_cams_list.py

2. Далее получаем URL для 20 камер:
Для запуска в терминале ввести:
python3 get_url.py


----- ДОПОЛНИТЕЛЬНАЯ ИНФОРМАЦИЯ О СКРИПТАХ -----
1. При запуске get_cams_list.py
- логи записываются в файл log_cam_list.log
- список камер записывается в файл response_cams_list.json

2. При запуске get_urls.py:
- рандомно генерируется:
    - дата начала поиска по кармерам (в диапазоне от 1 года до 3 суток до текущей даты )
    - дата окончания поиска по кармерам (в диапазоне от 1 секудны до 3 суток от начала поиска)
- логи и полученные URL записываются в файл log_url.log
